# Syncthing for IPFire

This plugin installs Syncthing on IPFire.

Tested on IPFire 2.29 (x86_64) Core Update 203

![](image/syncthing.png)

## Installation

```sh
sh install.sh
```
Run the installer as root from the project directory after copying it to your IPFire system.

## Uninstallation

```sh
sh uninstall.sh
```
